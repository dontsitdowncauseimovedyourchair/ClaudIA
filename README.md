# ClaudIA

A backend project that utilizes Open Router and the OpenAI SDK to create an agentic development Assistant that is able to work on code in a specified directory off of a single user prompt.

The best description I can give to this project is ✨**Claude Code from Temu.**✨

It's powered by a master system prompt and the handful of functions that I hand-wrote for essential capabilities like reading, writing and running files as well as listing directories. 

It is made with security in mind so it can only work in an assigned directory by having its function calls injected with the right working directory.
